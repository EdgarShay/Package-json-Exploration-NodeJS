# Package-json-Exploration-NodeJS Project

## Project Overview
This project demonstrates the structure and functionality of the `package.json`
file in a Node.js environment. It explores how package.json manages project metadata,
dependencies, scripts, versioning, and environment requirements.

The project includes a simple Node.js application and test cases designed to demonstrate dependency management and script automation.

---

## Learning Objectives
- Understand the purpose of `package.json`
- Manage dependencies and development dependencies using npm
- Create and run custom npm scripts
- Learn semantic versioning
- Understand the purpose of `package-lock.json`
- Define Node.js and npm version requirements using the engines field

---

## Technologies Used
- Node.js
- npm (Node Package Manager)
- Express.js
- Jest (Testing Framework)
